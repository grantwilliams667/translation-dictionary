# Translation Project
This project implements a single word translation system in C++ with two options which are hash table and binary search trees

The system works by reading the CSV file and translates the words from the file.

## Features

-CSV parsing
-English to spanish, italian, french, or german translation
-hash table implementation
-bst implementation
-runtime calculation

## Technologies
-c++
-hash tables
-File I/O
-BST
